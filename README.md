Desktop application built with PyQt5 and Qt Designer using Python for inventory management, the application allows CRUD operations and stores data using a JSON file.

Features:
Add new items to inventory
Edit existing items
Delete records
View all inventory data
Persistent storage using JSON

Technologies Used:
Python
PyQt5
Qt Designer
JSON

How to Run: 
Clone the repository: https://github.com/danielmorales27/PyQt5-app.git
Navigate to the project folder: cd pyqt5-app
Install dependencies: pip install -r requirements.txt
Run the application: python main.py

Project Structure:
│
├── main.py
├── ui/
│   └── diseño.ui
├── requirements.txt
└── README.md

Future Improvements:
